Алгоритм поиска подстроки (Задача 7.2)
В данном репозитории представлено решение задачи по поиску самой длинной подстроки без повторяющихся символов.

Программа принимает на вход массив произвольных данных и отфильтровывает строковые значения.

Используется алгоритм с применением срезов для эффективного поиска уникальной подстроки за один проход.

Код структурирован и разбит на три независимые функции: check_input, get_substring и main, что обеспечивает легкость чтения и тестирования.
